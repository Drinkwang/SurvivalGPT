## 🚀 快速开始

### 📦 安装依赖
```bash
# 自动安装（推荐）
./install.bat

# 手动安装
pip install -r requirements.txt
```

### 🎯 启动程序
```bash
# 使用启动脚本
./run.bat

# 直接运行
python main.py
```

### ⚙️ 配置AI模型
1. 点击界面上的"⚙️ 配置"按钮
2. 输入您的API密钥（DeepSeek、OpenAI等）
3. 点击"🧪 测试"验证连接
4. 在模型选择器中切换AI引擎
